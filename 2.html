📚 HTML基础

HTML是网页的骨架，定义了内容的结构和语义。

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <h1>主标题</h1>
    <p>段落内容</p>
</body>
</html>
🛠️ HTML标签

HTML常用标签聚合

<!-- 文档结构和元信息 -->
<!DOCTYPE html> <!-- 文档类型声明 -->
<html>        <!-- 根标签 -->
<head>        <!-- 文档头部 -->
    <meta charset="UTF-8"> <!-- 字符编码声明 -->
    <title>网页标题</title>
    <link rel="stylesheet" href="style.css"> <!-- 外部样式表 -->
    <script src="script.js"></script>        <!-- 外部脚本 -->
</head>
<body>        <!-- 网页主体内容 -->

<!-- 结构与布局 -->
<header> <!-- 页眉区域 -->
    <h1>网站标题</h1>
</header>
<nav> <!-- 导航区域 -->
    <ul>
        <li><a href="#home">首页</a></li>
        <li><a href="#about">关于</a></li>
    </ul>
</nav>
<main> <!-- 主要内容区 -->
    <article> <!-- 独立内容块，如文章 -->
        <h2>文章标题</h2>
        <p>文章内容...</p>
    </article>
    <section> <!-- 内容分区 -->
        <h3>分区标题</h3>
        <p>分区内容...</p>
    </section>
    <aside> <!-- 侧边栏内容 -->
        <p>侧边信息</p>
    </aside>
</main>
<footer> <!-- 页脚区域 -->
    <p>版权所有 © 2025</p>
</footer>

🎨 CSS样式

CSS用于控制网页的外观和布局。

<style>
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 20px;
}
h1 {
    color: #333;
    text-align: center;
}
</style>
🦄 CSS属性值

CSS属性示例

/* 布局与盒模型 */
display: 定义元素的显示类型 /* display: flex; */
position: 设置定位方式 /* position: absolute; */
top: 顶部偏移量 /* top: 10px; */
right: 右侧偏移量 /* right: 20px; */
bottom: 底部偏移量 /* bottom: 0; */
left: 左侧偏移量 /* left: 0; */
z-index: 层叠顺序 /* z-index: 100; */
float: 浮动方式 /* float: right; */
clear: 清除浮动影响 /* clear: both; */
width: 宽度 /* width: 200px; */
min-width: 最小宽度 /* min-width: 100px; */
max-width: 最大宽度 /* max-width: 400px; */
height: 高度 /* height: 150px; */
min-height: 最小高度 /* min-height: 50px; */
max-height: 最大高度 /* max-height: 300px; */
margin: 外边距 /* margin: 10px 20px; */
padding: 内边距 /* padding: 8px 16px; */
box-sizing: 盒模型计算方式 /* box-sizing: border-box; */
overflow: 内容溢出处理 /* overflow: auto; */
overflow-x: 水平方向溢出处理 /* overflow-x: scroll; */
overflow-y: 垂直方向溢出处理 /* overflow-y: hidden; */
gap: 项目间距（格栅/Flex布局） /* gap: 20px; *//* 边框与圆角 */
border: 边框综合设置 /* border: 1px solid #333; */
border-width: 边框宽度 /* border-width: 2px; */
border-style: 边框样式 /* border-style: dashed; */
border-color: 边框颜色 /* border-color: red; */
border-radius: 圆角半径 /* border-radius: 10px; */
outline: 轮廓线 /* outline: 2px solid blue; *//* 背景 */
background: 背景综合设置 /* background: #fff url(bg.jpg) no-repeat center; */
background-color: 背景色 /* background-color: #f8f8f8; */
background-image: 背景图片 /* background-image: url(bg.png); */
background-size: 背景尺寸 /* background-size: cover; */
background-repeat: 背景是否重复 /* background-repeat: no-repeat; */
background-position: 背景位置 /* background-position: right top; */
background-attachment: 背景滚动行为 /* background-attachment: fixed; *//* 字体与文本 */
color: 字体颜色 /* color: #222; */
font-size: 字体大小 /* font-size: 18px; */
font-family: 字体系列 /* font-family: Arial, sans-serif; */
font-weight: 字体粗细 /* font-weight: bold; */
font-style: 字体样式（斜体等） /* font-style: italic; */
text-align: 文字水平对齐方式 /* text-align: center; */
text-decoration: 文字装饰线 /* text-decoration: underline; */
line-height: 行高 /* line-height: 1.8; */
letter-spacing: 字符间距 /* letter-spacing: 2px; */
word-spacing: 单词间距 /* word-spacing: 8px; */
text-transform: 大小写转换 /* text-transform: uppercase; */
text-shadow: 文字阴影 /* text-shadow: 1px 1px 3px #999; */
white-space: 空白符处理方式 /* white-space: nowrap; */
direction: 文字方向 /* direction: rtl; *//* Flex布局 */
flex-direction: 主轴方向 /* flex-direction: column; */
flex-wrap: 是否换行 /* flex-wrap: wrap; */
flex-flow: 方向与换行 /* flex-flow: row wrap; */
justify-content: 主轴对齐方式 /* justify-content: space-between; */
align-items: 侧轴对齐方式 /* align-items: center; */
align-content: 多行侧轴对齐 /* align-content: stretch; */
order: 排序顺序 /* order: 2; */
flex-grow: 放大比例 /* flex-grow: 1; */
flex-shrink: 缩小比例 /* flex-shrink: 0; */
flex-basis: 初始大小 /* flex-basis: 100px; *//* Grid布局 */
grid-template-columns: 列轨道分布 /* grid-template-columns: 1fr 2fr; */
grid-template-rows: 行轨道分布 /* grid-template-rows: 100px 200px; */
grid-column: 元素跨越列数 /* grid-column: 1 / 3; */
grid-row: 元素跨越行数 /* grid-row: 2 / 4; */
grid-area: 区域名称或位置 /* grid-area: header; */
grid-gap: 网格间距 /* grid-gap: 16px; *//* 其他视觉样式 */
opacity: 不透明度 /* opacity: 0.6; */
box-shadow: 元素阴影 /* box-shadow: 2px 2px 8px #aaa; */
cursor: 鼠标形状 /* cursor: pointer; */
visibility: 可见性 /* visibility: hidden; */
transition: 过渡动画 /* transition: all 0.3s; */
transform: 变换（旋转/缩放等） /* transform: scale(1.1); */
animation: 动画属性 /* animation: fadeIn 1s ease-in; *//* 列表 */
list-style: 列表样式 /* list-style: square inside; */
list-style-type: 列表标记类型 /* list-style-type: decimal; */
list-style-image: 列表自定义图片标记 /* list-style-image: url(icon.png); *//* 表格 */
border-collapse: 边框合并 /* border-collapse: collapse; */
border-spacing: 单元格间距 /* border-spacing: 6px; */
table-layout: 表格布局算法 /* table-layout: fixed; *//* 伪类与伪元素 */
a:hover { color: red; } /* 鼠标悬停样式 */
button:active { background: #eee; } /* 按下时样式 */
.box::before { content: '★'; } /* 元素前插入内容 */
.box::after { content: '√'; } /* 元素后插入内容 *//* 响应式相关 */
@media (max-width: 600px) { ... } /* 媒体查询 *//* 其它常用属性 */
content: 伪元素内容 /* content: "Hello!"; */
pointer-events: 鼠标事件响应 /* pointer-events: none; */
user-select: 是否允许选中文本 /* user-select: none; *//* 更多高级/补充属性 */
backface-visibility: 背面是否可见 /* backface-visibility: hidden; */
filter: 图像/内容滤镜 /* filter: blur(3px); */
object-fit: 媒体填充方式 /* object-fit: cover; */
object-position: 媒体位置 /* object-position: center; */
resize: 用户是否能调整大小 /* resize: both; */
scroll-behavior: 滚动行为 /* scroll-behavior: smooth; */
will-change: 优化提示 /* will-change: transform; */
touch-action: 触摸行为控制 /* touch-action: none; */
clip-path: 裁剪路径 /* clip-path: circle(50%); */
mix-blend-mode: 混合模式 /* mix-blend-mode: multiply; */
writing-mode: 文字流向 /* writing-mode: vertical-rl; */
