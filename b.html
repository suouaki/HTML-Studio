📚 HTML基础

HTML是网页的骨架，定义了内容的结构和语义。

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <h1>主标题</h1>
    <p>段落内容</p>
</body>
</html>

🎨 CSS样式

CSS用于控制网页的外观和布局。

<style>
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 20px;
}
h1 {
    color: #333;
    text-align: center;
}
</style>

⚡ JavaScript交互

JavaScript为网页添加动态行为和交互功能。

<script>
function showAlert() {
    alert('Hello, World!');
}
</script>
